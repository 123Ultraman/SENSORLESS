仿真：
1、双击SENSORLESS.prj打开或关闭工程。工程包含两个模块。
2、一个是参数辨识，包括电阻、电感、磁链、转动惯量的辨识，电感、磁链、转动惯量的辨识结果在模型中直接输出，电阻的辨识结果需要在工程快捷方式中电机RsCalculate进行电阻计算。
3、另一个电机控制算法的仿真，包含磁链观测无感算法，弱磁控制和120度SVPWM算法和过调制算法，可以在模型中切换模式。
单片机程序：
1、程序在keil文件夹中，硬件为STM32G4 Nucleo-64 boards和X-NUCLEO-IHM07M1
2、上电后，先对进行ADC校准，然后等待按键。
3、短按按键实现对FOC运行模式的切换，依次为开环、速度闭环、双闭环和磁链观测器无感、过调制加弱磁控制。
4、长按按键进入电机参数辨识模式，依次辨别电阻、电感、磁链和转动惯量。
